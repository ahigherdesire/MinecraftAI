Getting Started

Up and running in 60 seconds.

Three steps from download to automation.

1

Drop in the JAR

Copy minecraftai-dirty.jar into your Fabric mods/ folder alongside Fabric Loader 0.18.4.

mods/minecraftai-dirty.jar
2

Activate Your Token

Paste your license token into Minecraft chat once. It's saved permanently — you'll never need to do it again.

#activate <your-token>
3

Start Commanding

Use any #command in chat. Type #help for the full list with tab completion on every argument.

#mine diamond_ore
